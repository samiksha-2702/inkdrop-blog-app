# 🖋️ InkDrop

InkDrop is a full-stack web application built with **Python, Django, Django REST Framework (DRF), REST APIs, HTML, CSS, JavaScript, and Bootstrap**. It enables users to create, edit, organize, and manage notes/blogs through a clean and responsive interface.

## 🚀 Features

- 🔐 User Authentication (Register/Login/Logout)
- ✍️ Create, Edit, and Delete Posts/Notes
- 📋 View and Manage All Posts
- 🔍 Search Functionality
- 📱 Responsive UI
- ⚡ REST API support using Django REST Framework

## 🛠️ Tech Stack

- **Backend:** Python, Django, Django REST Framework
- **Frontend:** HTML, CSS, JavaScript, Bootstrap
- **Database:** SQLite/MySQL
- **Version Control:** Git & GitHub

## ⚙️ Installation

1. Clone the repository:
```bash
git clone https://github.com/your-username/InkDrop.git
```

2. Navigate to the project:
```bash
cd InkDrop
```

3. Create and activate a virtual environment:
```bash
python -m venv venv
```

**Windows**
```bash
venv\Scripts\activate
```

**Linux/macOS**
```bash
source venv/bin/activate
```

4. Install dependencies:
```bash
pip install -r requirements.txt
```

5. Apply migrations:
```bash
python manage.py migrate
```

6. Run the development server:
```bash
python manage.py runserver
```

7. Open your browser and visit:
```
http://127.0.0.1:8000/
```
